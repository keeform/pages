Installation

Step 1: Add-on for KeePass

Download and run the KeeForm installer. The defaults are fine for most users. You only need to do this once, even if you install KeeForm on multiple browsers.
Add-on for KeePass

Step 2: Add-on for your browsers

Install the extension for each browser you want to use with KeeForm. You will be redirected to an external site.
Google Chrome
Mozilla Firefox
Microsoft Edge
Edge users: first enable Allow extensions from other stores in Edge Settings → Extensions.
If KeeForm stops launching Edge, disable Startup Boost in edge://settings/system.
After installing, make sure the extension is enabled — KeeForm shows a green indicator when everything is working. From now on, double-clicking an https URL in KeePass opens the browser and fills your login automatically.
